# 配置与快速开始

## 快速开始

```bash
# 1. 环境准备
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 2. 启动 Gateway（包含 Web UI）
python cli/main.py gateway --port 18790

# 3. 打开浏览器，进入 Web UI 进行配置
#    http://localhost:18790/ui
#    - 在「模型管理」中添加 LLM 提供商（API Key、模型名等）
#    - 在「通讯账号」中配置 IM 机器人（可选）
#    - 在「推送订阅」中配置通知渠道（可选）

# 4. 配置 Overleaf 同步（可选）
pip install overleaf-sync
ols login  # 生成 .olauth 认证文件

# 5. 或者使用 CLI 模式交互
python cli/main.py agent
```

> **提示**：所有配置通过 Web UI 完成即可。配置最终存储在 `settings.json` 中，高级用户也可以直接编辑该文件。

## 配置文件说明

| 文件 | 用途 |
| --- | --- |
| `settings.json` | 统一运行时配置（LLM 提供商、IM 账号、推送、网关等），通过 Web UI 管理 |
| `config/tools.json` | 工具注册表（启用/禁用、模式授权、角色限制） |
| `config/agents.json` | SubAgent 全局配置（merge_to_core 等） |
| `config/commands.json` | 斜杠命令定义 |
| `workspace/{project}/project.yaml` | 项目级配置（Git、Overleaf、LaTeX 引擎） |

## 系统诊断

如需检查配置是否正确，可运行：

```bash
python cli/main.py doctor
```
