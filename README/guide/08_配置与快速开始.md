# 配置与快速开始

## 配置文件说明

| 文件                                 | 用途                                                                        |
| ------------------------------------ | --------------------------------------------------------------------------- |
| `settings.json`                    | 统一配置（`provider.instances`、`channel.accounts`、gateway、全局参数） |
| `config/features.json`             | 功能开关（history、memory、agent 参数）                                     |
| `config/tools.json`                | 工具注册表（启用/禁用、模式授权、角色限制）                                 |
| `config/agents.json`               | SubAgent 全局配（merge_to_core 等）                                       |
| `config/commands.json`             | 斜杠命令定义                                                                |
| `config/modes.json`                | 模式定义（CHAT/NORMAL/TASK/TEAM）                                           |
| `workspace/{project}/project.yaml` | 项目级配置（git、overleaf、latex）                                          |

## 快速开始

```
# 1. 环境准备
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 2. 初始化配置（推荐）
python cli/main.py onboard

# 3. 如需手动检查，运行系统诊断
python cli/main.py doctor

# 4. 配置 Overleaf（可选）
ols login  # 生成 .olauth

# 5. 启动 CLI
python cli/main.py agent

# 6. 或启动 Gateway + Web UI
python cli/main.py gateway --port 18790

在Web UI配置 http://127.0.0.1:port/ui
```
